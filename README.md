Lab Task 6: Two-Player Reaction Game
Course: ENCE 3608 - 20291 Embedded Systems
ADA university
Author [Nabiyev Rovshan]
Date[May,2026]
Project Overview:
An interactive two-player reaction time game combining Arduino hardware with a Python GUI host application. Players compete in best-of-5 matches by pressing buttons as fast as possible after an LED signal, with all performance data logged and visualized in real-time.
